# Nestcode Website

Website statis sederhana untuk portofolio dan layanan jasa coding "Nestcode".

## Isi Proyek

- `index.html` - halaman utama website.
- `styles.css` - semua gaya dan tata letak halaman.
- `script.js` - interaksi menu mobile dan animasi reveal.
- `vercel.json` - konfigurasi deployment statis untuk Vercel.

## Fitur

- Navbar dengan logo dan menu navigasi responsif.
- Tombol kontak yang langsung membuka email dan WhatsApp.
- Tampilan responsif untuk desktop dan mobile.
- Animasi muncul saat elemen terlihat di layar.

## Menjalankan Lokal

Cukup buka `index.html` di browser untuk melihat hasilnya secara lokal.

## Deploy ke Vercel

1. Pastikan punya akun Vercel.
2. Jalankan di folder proyek:

```bash
vercel login
vercel
```

3. Atau deploy dari GitHub repository dengan menghubungkan project ke Vercel.

## Catatan

`vercel.json` sudah dikonfigurasi untuk deploy sebagai situs statis dengan `@vercel/static`.
